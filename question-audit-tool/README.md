# 問題データ監査ツール

## 概要

麻雀点数計算トレーナーで利用している
questions.js を監査するためのツールです。

本ツールはアプリ本体とは独立した
開発者向けツールです。

---

## ファイル構成

```
question-audit-tool/

index.html

styles.css

audit.js

questions.js
```

questions.js は本体からコピーしてください。

---

## Version0.1

実装内容

- questions.js読み込み
- 問題件数表示
- 問題一覧表示
- 監査ボタン
- 監査結果表示

監査処理はVersion0.2で実装予定です。
