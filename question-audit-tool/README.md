# 問題データ監査ツール

## 概要

麻雀点数計算トレーナーで利用する `questions.js` を監査するための補助ツールです。

`app/questions.js` を `question-audit-tool/questions.js` にコピーして利用します。

## Version 0.6

監査項目は次の9項目です。

1. 問題ID重複チェック
2. 必須項目チェック
3. 翻数整合性チェック
4. 符整合性チェック
5. 点数カテゴリ整合性チェック
6. 切り上げ満貫整合性チェック
7. 役の翻数合計チェック
8. 役一覧整合性チェック
9. 点数カテゴリ計算チェック

## 点数カテゴリ計算チェック

`answer.fu` と `answer.totalHan` から、麻雀の点数計算ルールに従って
点数カテゴリを算出し、`answer.score.category` と比較します。

判定するカテゴリ:

| 条件 | カテゴリ |
|---|---|
| 通常点 | 通常 |
| 30符4翻・60符3翻 | 切り上げ満貫 |
| 5翻、または基礎点が2000以上 | 満貫 |
| 6～7翻 | 跳満 |
| 8～10翻 | 倍満 |
| 11～12翻 | 三倍満 |
| 13翻以上 | 数え役満 |

例として50符4翻は、基礎点が2000以上になるため「満貫」です。

不一致時の表示例:

```text
q106: 50符4翻 / 計算したカテゴリ="満貫" / answer.score.category="通常"
```

## 起動方法

1. `app/questions.js` を監査ツールのフォルダへコピーする
2. `index.html` をブラウザで開く
3. 「監査開始」を押す

- Ver0.7: 最終点数表示チェック

- Ver1.0: 最終整理
