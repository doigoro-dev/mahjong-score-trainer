# 🀄 Mahjong Score Trainer

> **実戦形式で麻雀の点数計算を学べるWebアプリ**

麻雀の点数計算は、多くのプレイヤーが最初につまずくポイントです。

本アプリは、

- 役判定
- 翻数計算
- 符計算
- 点数計算

を段階的に学習し、最終的には実戦に近い流れで自然に点数計算ができるようになることを目的として開発しました。

初心者の方はもちろん、

「点数計算は何となく分かるけど、まだ時間がかかる」

という中級者の方にも役立つ学習アプリです。

# 🎮 Learning Modes

Version2では、目的に応じて4種類の学習モードを用意しています。

---

## ① Normal Mode

もっとも基本となる学習モードです。

表示された問題から、

- 役
- 翻数
- 符
- 点数

を順番に回答します。

### こんな人におすすめ

- 点数計算を初めて学ぶ
- 基本を繰り返し練習したい
- 役を覚えながら学びたい

---

## ② Han Variation Mode

同じ手牌でも、

- リーチ
- 一発
- ドラ
- 裏ドラ

などによって翻数がどのように変化するかを学習できます。

同じ問題を様々な条件で解くことで、

翻数計算を素早く行えるようになります。

### こんな人におすすめ

- ドラ計算が苦手
- リーチ時の計算に慣れたい
- 翻数の変化を理解したい

---

## ③ Practical Mode

Version2最大の特徴です。

実際の対局に近い流れで問題が出題されます。

### 学習の流れ

```
問題表示

        ↓

リーチする？

        ↓

ツモ？ロン？

        ↓

ドラ表示

        ↓

裏ドラ（リーチ時）

        ↓

役判定

        ↓

翻数

        ↓

符計算

        ↓

点数計算

        ↓

自動採点
```

知識だけではなく、

**実戦で必要となる思考順序**

を身につけられることが、このモード最大の特徴です。

### こんな人におすすめ

- 実戦で点数計算が遅い
- 点数計算を習慣化したい
- 対局中でも素早く計算したい

---

## ④ Question Selection / Review Mode

収録されている問題を自由に選択できます。

また、

間違えた問題だけを後から復習することも可能です。

苦手問題を重点的に学習できるため、

効率よく実力を伸ばせます。

### 主な用途

- 苦手問題の復習
- 特定の問題を再確認
- 実戦前のウォーミングアップ

---

# 🚀 Learning Flow

初めて利用する場合は、次の順番がおすすめです。

1. Normal Mode

基礎から実戦まで、段階的に学習できるよう設計されています。
