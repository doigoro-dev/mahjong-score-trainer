1. アプリの概要

麻雀点数計算トレーナーは、実戦に近い牌姿を見ながら、 役・翻数・符数・最終的な点数を自分で計算するための学習アプリです。

本アプリは練習用です。
対局画面やスクリーンショットを読み取って、自動的に点数を判定するアプリではありません。
収録問題

全100問

1回のセッションで、100問が重複なしで出題されます。

主な目的

点数計算への習熟

役判定だけでなく、符計算から最終点数までを一連で練習します。

Ver1.6では、通常モードに加えて、立直・一発・ドラなどの条件を追加した 「翻数変化モード」を利用できます。

2. 利用開始方法

  1. アプリ本体のHTMLファイルを、Chrome、Edge、Safariなどのブラウザで開きます。
  2. 最初の問題が自動的に表示されます。インストールやログインは不要です。
  3. 手牌・和了方法・場風・自風を確認し、役・翻・符・点数を計算します。
パソコン上のHTMLファイルをダブルクリックするだけでも利用できます。 公開後は、配布されたURLへアクセスするだけで利用できます。

3. 基本的な使い方

  1. 画面上部で現在の問題番号と残り問題数を確認します。
  2. 手牌と和了牌を見て、成立する役を考えます。
  3. 和了方法（ロン／ツモ）、場風、自風を確認します。
  4. 符数、翻数、親子の別を踏まえて、最終的な点数を計算します。
  5. 計算できたら「答えを見る」ボタンを押します。
  6. 正解内容を確認した後、「次の問題へ」で進みます。
アプリ上で回答を入力する形式ではありません。 まず頭の中やメモで計算してから、答えを表示して確認してください。

4. 出題モード

基本

通常モード

表示された牌姿から、基本の役・翻・符・点数を計算します。

立直・一発・ドラ・赤ドラ・裏ドラ・副露は、基本的に問題条件として扱いません。

応用

翻数変化モード

元の牌姿に加えて、画面に表示された付加条件を加味して計算します。

各問題には複数の候補パターンがあり、セッション開始時にその中から 1つがランダムに選ばれます。

翻数変化モードで表示される条件

  • 立直
  • 一発
  • ドラ
  • 赤ドラ
  • 裏ドラ
計算方法
牌姿から読み取れる基本翻数に、表示された付加条件の翻数を足してください。 符数、親子、ロン／ツモは元の問題条件を使用します。

例：

基本：40符2翻 ＋ 立直1翻・ドラ2翻 → 40符5翻

モード切り替え時の動作

モードを切り替えると確認ダイアログが表示されます。 確認後は現在の進捗を破棄し、選択したモードの第1問から開始します。

5. 答えの見方

「答えを見る」を押すと、主に次の情報が表示されます。

成立役 その問題で成立する役と、それぞれの翻数です。
合計翻数 基本役と付加条件を合計した翻数です。
符数 最終的に使用する符数です。
点数区分 通常、満貫、跳満、倍満、三倍満、数え役満などを表示します。
支払い点数 親子およびロン／ツモに応じた最終点数です。
符の内訳 副底、面子、待ち、門前ロン、ツモなどの計算根拠です。
間違えた場合は、点数だけでなく「どの役を見落としたか」「符のどこで差が出たか」 まで確認すると、学習効果が高まります。

6. 進捗保存・途中再開・やり直し

途中再開

問題の順番、現在位置、答えの表示状態、翻数変化モードの付加条件は、 ブラウザのセッション保存領域に記録されます。

同じブラウザタブを再読み込みした場合は、原則として続きから再開します。

最初からやり直す

「最初からやり直す」ボタンを押すと確認ダイアログが表示されます。 OKを押すと、現在の進捗を破棄し、問題順を再シャッフルして第1問から開始します。

翻数変化モードでは、各問題の付加条件パターンも再選択されます。

100問終了後

全100問を終了すると完了画面が表示されます。 「もう一度挑戦する」から新しいセッションを開始できます。

ブラウザの閲覧データやサイトデータを削除した場合、 保存されていた進捗が消えることがあります。

7. スマートフォン・狭い画面での利用

Ver1.6では、表示領域に合わせて牌の大きさと間隔が自動調整されます。 原則として、牌を見るための横スクロールは発生しません。

  • PCでは、見やすい最大サイズで表示します。
  • タブレットでは、画面幅に応じてなめらかに縮小します。
  • スマートフォンでは、14枚全体が収まるように縮小します。
  • 和了牌との間隔や「和了牌」ラベルも連動して調整します。
画面が極端に狭い場合は、端末を横向きにすると牌をより大きく表示できます。

9. 現時点で対象外の機能・注意事項

  • 副露を含む問題
  • 食い下がりの判定
  • 明刻・暗刻の状態が付加条件によって変化する問題
  • 実際の対局画像やスクリーンショットの自動認識
  • 回答入力による正誤判定
  • 正答率や苦手問題の自動集計
  • ローカルルールすべてへの対応
麻雀のルールや点数計算には、団体・雀荘・ゲームごとの差異があります。 本アプリの表示と、普段利用しているルールが異なる場合があります。

また、本バージョンはベータ版です。問題データや表示内容に誤りが残っている可能性があります。 実戦の精算など、正確性が必須の用途では、別の点数表や公式ルールも併せて確認してください。

10. 感想・不具合・改善要望の伝え方

ベータ版では、実際に利用した方からの感想を基に改善を進めます。 次の情報があると、内容を確認しやすくなります。

  • 問題番号：例「第35問」
  • 利用モード：通常モード／翻数変化モード
  • 利用端末：PC、iPhone、Androidなど
  • 利用ブラウザ：Chrome、Edge、Safariなど
  • 発生した内容：何をした時に、何が起きたか
  • 期待する動作：本来どう表示・動作してほしかったか
  • 可能であれば：画面のスクリーンショット

感想をもらう際の質問例

  1. 問題の難易度は適切でしたか。
  2. 分かりにくい表示や操作はありましたか。
  3. 間違えた問題の復習機能は欲しいですか。
  4. 追加してほしい出題モードはありますか。
  5. 今後も利用したいと思いましたか。
