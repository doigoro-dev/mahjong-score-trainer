1. アプリの概要

麻雀点数計算トレーナーは、牌のイラストを見ながら、 役・翻数・符数・最終的な点数を自分で判断するための学習アプリです。

Version 2では、通常の問題演習に加えて、 立直判断、和了方法、ドラ表示、裏ドラ確認を順に進める 実戦モードを利用できます。

本アプリで特に重視しているのは、 答えだけを覚えることではなく、 実戦に近い思考順序で点数を考えることです。

主な機能

  • 全117問を収録
  • 通常モード
  • 翻数変化モード
  • 実戦モード
  • 問題選択・復習モード
  • タイマーによる解答時間の計測
  • 入力した点数の自動採点
  • 役・翻・符・点数の解説表示
  • セッションの保存と再開
  • 復習中の進行状況保持
  • PC・スマートフォン対応

学習対象

役判定

牌姿と条件を見て、成立している役を確認します。

翻数計算

基本役、立直、ドラ、裏ドラなどを合計します。

符計算

副底、和了方法、面子、雀頭、待ちなどを確認します。

点数計算

親・子、ロン・ツモ、符・翻数から最終点数を求めます。

3. 学習の進め方

初めて利用する場合は、次の順番で進めると、 基本から実戦形式まで段階的に練習できます。

  1. 通常モード
    手牌、親子、ロン・ツモなどの基本条件から、 役・翻・符・点数を計算します。
  2. 翻数変化モード
    立直、一発、ドラ、赤ドラ、裏ドラなどの付加条件を含めて計算します。
  3. 実戦モード
    表ドラを確認し、立直判断、和了結果、裏ドラ確認を経て最終点数を入力します。
  4. 問題選択・復習モード
    間違えた問題や確認したい問題を選び、繰り返し練習します。

問題を解く時の基本順序

牌姿・条件を確認 → 成立役を確認 → 翻数を合計 → 符を計算 → 点数を決定
慣れるまでは時間を気にせず、役と符の根拠を確認してください。 慣れてきたらタイマーを目安に、判断時間を短くしていきます。

4. 学習モード

通常モード

手牌と基本条件から、役・翻・符・点数を計算します。 点数計算の基本練習に適しています。

翻数変化モード

基本の牌姿へ立直、一発、ドラ、赤ドラ、裏ドラなどの条件を追加し、 翻数と点数の変化を練習します。

実戦モード

表ドラ確認、立直判断、和了結果、裏ドラ確認という順序で進み、 実戦に近い流れで最終点数を計算します。

問題選択・復習モード

問題一覧から任意の問題を選択したり、 復習対象の問題を続けて解いたりできます。

モード切替時の注意

モードを切り替えると、現在の通常セッションを破棄して 第1問から開始する旨の確認ダイアログが表示される場合があります。 継続中の問題がある場合は、切り替える前に内容を確認してください。

5. 実戦モード

実戦モードは、Version 2の中心となる学習モードです。 最初から全条件を表示するのではなく、 実際の対局で情報を確認する順序に近い形で進みます。

基本フロー

牌姿・表ドラを確認 → 立直するか選択 → ツモ・ロンを確認 → 裏ドラを確認 → 最終点数を入力 → 自動採点・解説

操作手順

  1. 手牌、場風、自風、親子、表ドラ表示牌など、 画面に表示された情報を確認します。
  2. 現在の牌姿と打点を考え、立直するかどうかを選択します。
  3. 画面の案内に従って、ツモまたはロンの和了結果を確認します。
  4. 立直して和了した場合は、裏ドラ表示牌を確認します。
  5. 役、翻、符を考え、最終的な点数を入力します。
  6. 採点結果と解説を確認し、次の問題へ進みます。

立直・ドラの扱い

  • 立直を選択した場合、立直は最終的な翻数へ反映されます。
  • 表ドラは、立直の有無にかかわらず確認します。
  • 裏ドラは、立直して和了した場合に確認します。
  • ドラおよび裏ドラは、表示牌そのものではなく次の牌がドラです。

役なしロンの扱い

立直しない状態でロンすると役が成立しない問題では、 実戦モードの進行を成立させるため、 和了方法がツモへ自動変換される場合があります。

Version 2では、暗槓を含む問題でも、 表ドラ表示牌および裏ドラ表示牌は各1枚固定です。 槓子数に応じた槓ドラ・槓裏ドラの追加表示は対象外です。

6. 問題選択・復習モード

問題選択・復習モードでは、 全問題を順番に解く方法とは別に、 必要な問題へ直接移動して練習できます。

問題選択

  • 問題一覧から任意の問題を選択できます。
  • 特定の役・符・翻数を含む問題を後から確認する用途に利用できます。
  • 一度解いた問題を再確認する場合にも利用できます。

復習モード

  • 採点で間違えた問題を復習対象として扱います。
  • 復習対象の問題だけを続けて解くことができます。
  • 復習中の進行状況は保持されます。

おすすめの使い方

  1. 通常モードまたは実戦モードで問題を解きます。
  2. 間違えた問題では、役・符・点数の解説を確認します。
  3. 学習後に復習モードを開きます。
  4. 同じ問題を、自分で説明できる状態になるまで解き直します。
正解の点数だけでなく、 「どの役が成立し、どこで符が増えたか」を説明できるか確認すると、 実戦で再現しやすくなります。

7. 解答・採点の見方

点数入力

回答欄へ自分が計算した点数を入力し、採点ボタンを押します。 ツモ問題では、親・子または子・子の支払額を画面の形式に合わせて入力します。

採点結果

正解 入力した点数と登録された正解が一致しています。
不正解 正解の点数と異なります。表示された解説を確認します。
未入力・形式エラー 必要な欄が空欄、または数値として扱えない場合に案内が表示されます。

解説で確認できる内容

成立役 牌姿と条件から成立する役、および各役の翻数
合計翻数 基本役、立直、ドラ、裏ドラなどを合計した翻数
符数 最終的な符数
符の内訳 副底、門前ロン、ツモ、雀頭、待ち、刻子、暗槓など
点数 親・子、ロン・ツモを反映した最終点数

タイマー

問題画面には経過時間が表示されます。 タイマーは、正解・不正解を決めるものではなく、 判断速度を確認するための目安です。

最初は正確さを優先し、 計算方法が安定してから時間短縮を意識してください。

8. 保存・再開

通常セッションの復元

問題の出題順、現在位置、回答表示状態、利用モード、 実戦モードの進行状態などがブラウザ内へ保存されます。 同じブラウザ環境で再度開くと、途中から再開できます。

復習進行状況の保持

復習モードで現在どの問題まで進んでいるかも保持されます。 一度にすべて解けない場合でも、後から続きを進められます。

最初からやり直す

「最初からやり直す」操作を行うと、 現在の通常セッションを破棄し、 問題順を改めて設定して第1問から開始します。 実行前に確認ダイアログが表示されます。

ブラウザのサイトデータ、ストレージ、閲覧データを削除すると、 保存されたセッションや復習情報が失われる場合があります。 別のブラウザや別の端末へ自動的には引き継がれません。

9. スマホでの利用

Version 2はレスポンシブ表示に対応しています。 PC、タブレット、スマートフォンの画面幅に合わせて、 牌や操作領域の大きさが調整されます。

牌表示

  • 通常の牌姿は、画面幅に合わせて自動的に縮小されます。
  • 暗槓を含む問題では、暗槓の組数に応じて牌サイズが調整されます。
  • 横スクロールを極力使わず、問題全体を確認できるようにしています。

操作上の注意

  • 画面を横向きにすると、牌や解説を見やすくなる場合があります。
  • ブラウザの文字サイズを大きくしている場合、表示が折り返されることがあります。
  • 入力欄では、端末によって数値キーボードの表示が異なります。

10. 対応ルール

Version 2で扱う主な条件と役は次のとおりです。 全問題にすべての役が含まれるわけではありません。

和了方法 ロン、ツモ
親子 親、子
風 場風、自風
基本役 立直、門前清自摸和、平和、断么九、一盃口、二盃口、 役牌、七対子、対々和、三暗刻など
順子系・色系など 三色同順、一気通貫、混全帯么九、純全帯么九、 混一色、清一色、小三元、混老頭、三色同刻など
ドラ関連 ドラ、赤ドラ、裏ドラ
槓 暗槓、複数暗槓
符計算 基本符、門前ロン、ツモ符、雀頭、待ち、刻子、暗槓、 平和ツモ、七対子25符、10符単位への切り上げ
点数区分 通常、切り上げ満貫、満貫、跳満、倍満、三倍満、数え役満
切り上げ満貫 30符4翻、60符3翻を満貫として扱います。

Version 2での槓の扱い

暗槓および複数暗槓を含む問題を収録しています。 暗槓による符は合算して採点されます。

一方で、槓子数に応じた表ドラ表示牌・裏ドラ表示牌の追加は行いません。 Version 2では各1枚固定です。

11. FAQ

Q. 途中でブラウザを閉じても続きから再開できますか。

同じブラウザ環境で保存情報が残っていれば、途中から再開できます。

Q. 1回のセッション中に同じ問題は何度も出ますか。

通常セッションでは、全117問を一巡するまで、 同じ問題を重複なく出題します。

Q. 間違えた問題だけを解き直せますか。

問題選択・復習モードから復習できます。

Q. スマートフォンでも利用できますか。

利用できます。画面幅に合わせて牌と操作領域が調整されます。

Q. 表ドラ表示牌とドラは同じ牌ですか。

同じ牌ではありません。基本的には、表示牌の次の牌がドラです。

Q. 実戦モードで立直しなかったのに、ロンがツモへ変わりました。

立直なしのロンでは役が成立しない問題について、 和了を成立させるためにツモへ自動変換される場合があります。

Q. 暗槓があるのにドラ表示牌が増えないのはなぜですか。

Version 2では、暗槓問題でも表ドラ・裏ドラ表示牌を各1枚固定としています。

Q. 学習データはサーバーへ送信されますか。

セッションや復習情報は、基本的に利用しているブラウザ内へ保存されます。

12. 対象外・注意事項

Version 2で対象外の主な機能

  • チー、ポン、大明槓、加槓などの副露問題
  • 副露による食い下がりの判定
  • 槓子数に応じた槓ドラ・槓裏ドラの複数表示
  • 実際の対局画像やスクリーンショットの自動認識
  • 外部麻雀アプリからの画像読み込み
  • クラウドへの学習データ同期
  • オンラインランキング
  • すべてのローカルルールへの対応
麻雀の役、符、点数計算には、 競技団体、雀荘、ゲーム、地域によるルール差があります。 本アプリの結果と、普段利用しているルールが異なる場合があります。

学習上の注意

本アプリは点数計算の練習用です。 実際の対局では、供託、積み棒、責任払い、ローカル役など、 本アプリで扱っていない条件が加わる場合があります。

13. 感想・不具合報告

表示、採点、問題データ、操作方法などに不具合がある場合は、 再現に必要な情報を添えて報告してください。

報告時にあると分かりやすい情報

  • 問題番号：例「第35問」「q035」
  • 利用モード：通常、翻数変化、実戦、問題選択・復習
  • 利用端末：PC、iPhone、Androidなど
  • 利用ブラウザ：Chrome、Edge、Safariなど
  • 操作手順：どの操作をした時に発生したか
  • 実際の結果：何が表示・実行されたか
  • 期待する結果：本来どうなると考えたか
  • スクリーンショット：可能な場合のみ

問題内容の報告

役、符、翻数、点数、ドラ、解説に誤りがあると思われる場合は、 問題番号と、誤りと思われる箇所を記載してください。
